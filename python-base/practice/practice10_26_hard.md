# Проверочная работа по Python (10-26 урок)

## Тема: Обработка данных студентов

---

## Цель работы

Разработать консольную программу на Python, которая обрабатывает данные о студентах, используя:

* строки и их методы;
* списки и вложенные списки;
* словари;
* циклы `while` и `for`;
* условные операторы `if / elif / else`;
* операторы `break`, `continue`;
* ввод данных через `input()` и вывод через `print()`.

⚠️ **Функции использовать запрещено.**
Весь код должен быть написан линейно, с использованием только изученных конструкций.

---

## Входные данные

Программа работает с **одной многострочной строкой**, содержащей информацию о студентах.

### Формат одной строки

```
ФИО студента; группа; предмет; оценки
```

* **ФИО** — строка
* **Группа** — `rpo1`, `rpo2`, `rpo3`
* **Предмет** — `html`, `python`, `oop`, `javascript`, `django`
* **Оценки** — числа от 4 до 12, перечисленные через запятую

### Пример

```
Иванов Иван Иванович; rpo1; html; 8,10,9,10,11
```

Один студент может встречаться в данных **несколько раз**, если у него несколько предметов.

---

## Общая структура программы

Программа представляет собой **меню**, работающее в бесконечном цикле, пока пользователь не выберет выход.

---

## Основные этапы реализации (обязательны)

### Этап 1. Подготовка и анализ данных

1. Создать одну многострочную строку с данными студентов.
2. Разбить строку на отдельные строки.
3. Для каждой строки:

   * распарсить данные;
   * извлечь ФИО, группу, предмет и оценки;
   * привести оценки к числам.

---

### Этап 2. Создание структуры данных

На основе распарсенных данных необходимо создать структуру:

```python
students = {
    "Иванов Иван Иванович": {
        "group": "rpo1",
        "subjects": {
            "html": [8, 10, 9, 10, 11],
            "python": [9, 9, 10, 10, 11]
        }
    }
}
```

Разрешено использовать только:

* словари;
* списки;
* вложенные структуры.

---

### Этап 3. Основное меню программы

При запуске программа должна выводить меню:

```
1. Загрузить данные
2. Вывести данные о группе
3. Вывести данные одного студента
4. Добавить нового студента
5. Удалить студента
0. Выход
```

Программа должна работать в цикле `while`.

---

## ЭТАП 1. Базовый функционал

### Пункт 1. Загрузить данные

* Выполняется парсинг строки.
* Создается словарь `students`.
* При повторной загрузке данные **перезаписываются**.
* После загрузки выводится сообщение об успешной операции.

---

### Пункт 2. Вывести данные о группе

1. Пользователь вводит название группы.
2. Программа выводит **ФИО всех студентов** этой группы.
3. Если студентов нет — вывести соответствующее сообщение.

---

### Пункт 3. Вывести данные одного студента

1. Пользователь вводит **фамилию**.
2. Программа ищет всех студентов, у которых фамилия совпадает.
3. Если совпадений несколько — вывести всех.
4. Если нет — вывести сообщение.

После выбора студента открывается **дополнительное меню**.

---

## ЭТАП 2. Дополнительное меню студента

```
1. Показать оценки
2. Показать средний балл по предмету
0. Назад
```

### Пункт 1. Показать оценки

* Вывести все предметы студента.
* Для каждого предмета вывести список оценок.

---

### Пункт 2. Средний балл по предмету

1. Пользователь вводит название предмета.
2. Возможные варианты:

   * предмет отсутствует → `Нет такого предмета`
   * оценок < 5 → `Мало оценок`
   * оценок ≥ 5 → вывести среднее арифметическое

---

## ЭТАП 3. Работа с данными

### Пункт 4. Добавление нового студента

1. Пользователь вводит:

   * ФИО
   * группу
   * предмет
   * оценки
2. Если студент уже существует:

   * добавляется новый предмет
3. Если нет:

   * создается новая запись

---

### Пункт 5. Удаление студента

1. Пользователь вводит фамилию.
2. Если найден один студент — удалить.
3. Если несколько — вывести список и удалить всех с совпадением.
4. Если не найден — вывести сообщение.

---

## Завершение программы

* При вводе `0` в основном меню программа завершается.

---

## Дополнение к условию

> ⚠️ Один студент может иметь несколько предметов, но группа у него всегда одна.
