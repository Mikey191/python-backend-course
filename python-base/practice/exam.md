# Экзаменационная работа

## Тема: Обработка файлов, функции, исключения, модули и декораторы

### Описание задачи

Необходимо разработать программу для анализа списка электронных адресов, хранящихся в текстовом файле.

Программа должна прочитать файл с email-адресами, проверить корректность каждого адреса, отфильтровать нужные данные и сохранить результат в отдельный файл.

---

## Исходные данные

Создайте текстовый файл `emails.txt`.

Каждый email должен располагаться на отдельной строке.

Пример содержимого файла:

```text
ivan@gmail.com
petrov@yandex.ru
admin@mail.ru
test@test.com
user123@company.ru
invalid_email
example.ru
user@gmail
test@@mail.ru
hello@mail.org
student@university.com
support@service.ru
```

---

## Требования к программе

### 1. Чтение данных

Необходимо прочитать все строки из файла `emails.txt`.

Для каждого email выполнить проверку корректности.

---

### 2. Проверка email

Реализуйте следующие проверки:

#### Проверка №1

Email должен содержать символ `@`.

Если символ отсутствует — необходимо выбросить исключение.

Пример некорректного адреса:

```text
example.ru
```

---

#### Проверка №2

Email должен заканчиваться на допустимую доменную зону.

Допустимые варианты:

```python
(".ru", ".com", ".org", ".net")
```

Если адрес заканчивается на другую строку или не имеет доменной зоны — необходимо выбросить исключение.

Примеры некорректных адресов:

```text
user@gmail
example@test
```

---

### 3. Общая функция валидации

Создайте отдельную функцию, которая вызывает обе проверки.

Если хотя бы одна проверка не пройдена, должно возникать исключение.

---

### 4. Обработка ошибок

При обработке списка email используйте конструкцию:

```python
try:
    ...
except:
    ...
```

Если email не прошёл проверку:

* программа не должна завершаться;
* необходимо вывести сообщение об ошибке в терминал;
* программа должна продолжить обработку остальных строк.

Пример сообщения:

```text
Ошибка: invalid_email не содержит символ '@'
```

---

### 5. Фильтрация данных

После успешной проверки необходимо оставить только email-адреса, которые относятся к домену `.ru`.

Пример:

Оставляем:

```text
admin@mail.ru
user@company.ru
```

Не оставляем:

```text
user@gmail.com
hello@mail.org
```

---

### 6. Сохранение результата

Все найденные email с доменом `.ru` необходимо записать в файл:

```text
filtered_emails.txt
```

Каждый адрес должен быть записан на новой строке.

---

# Задание повышенной сложности №1

Разделите программу на модули.

Пример структуры проекта:

```text
project/

├── emails.txt
├── main.py
├── file_utils.py
├── validators.py
├── app.py
└── decorators.py
```

### Назначение модулей

`file_utils.py`

* чтение файла;
* запись файла.

`validators.py`

* все функции проверки email.

`app.py`

* основная логика приложения.

`decorators.py`

* пользовательские декораторы.

`main.py`

* запуск программы.
