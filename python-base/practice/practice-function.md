# «Анализ посещаемости мероприятий»

---

## 1. Описание задачи

Вам необходимо реализовать консольную программу для анализа посещаемости мероприятий.

Исходные данные представлены в виде **строк фиксированного формата**:

```
НазваниеМероприятия;ИмяФамилия;ИмяФамилия;ИмяФамилия;
```

* первое значение — **название мероприятия**
* далее — **список гостей**
* максимум **10 гостей**
* имена гостей **могут повторяться**
* один и тот же гость может посещать разные мероприятия

Программа должна:

* разобрать (распарсить) входные строки;
* сохранить данные в удобной структуре;
* позволять добавлять новые мероприятия и гостей;
* выполнять аналитические операции.

---

## 2. Требования к структуре программы

1. Вся логика должна быть **разбита на функции**
2. Программа должна запускаться из функции `main()`
3. В `main()` должен быть **бесконечный цикл** с меню
4. Разрешено использовать:

   * `list`, `dict`, `set`, `tuple`
   * циклы `for`, `while`
   * функции с аргументами и `return`
5. **ООП, файлы, сторонние библиотеки запрещены**

---
