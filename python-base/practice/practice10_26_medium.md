# Проверочная работа по Python (10-26 урок облегчённый уровень)

## Тема: Работа со студентами и их оценками

---

## Цель задания

Создать консольную программу для хранения и обработки данных о студентах, используя **только базовые конструкции Python**, которые уже были изучены:

* строки и методы строк;
* списки;
* словари;
* циклы `while`, `for`;
* условные операторы `if / elif / else`;
* ввод данных `input()` и вывод `print()`.

⚠️ **Использование функций запрещено.**
Вся логика реализуется линейно, без `def`.

---

## Формат входных данных

Программа работает с **одной многострочной строкой**, в которой каждая строка описывает одного студента.

### Формат одной строки

```
ФИО студента; группа; оценки
```

* **ФИО** — строка
* **Группа** — `rpo1`, `rpo2`, `rpo3`
* **Оценки** — числа от 4 до 12, перечисленные через запятую

### Пример

```
Иванов Иван Иванович; rpo1; 8,9,10,9,11
```

---

## Структура данных

После загрузки данных необходимо сформировать словарь следующего вида:

```python
students = {
    "Иванов Иван Иванович": {
        "group": "rpo1",
        "marks": [8, 9, 10, 9, 11]
    }
}
```

---

## Основные этапы решения (рекомендуется разобрать заранее)

### 1. Создание меню программы

* Использовать бесконечный цикл `while True`
* Реализовать выход из программы при выборе `0`

---

### 2. Загрузка и парсинг данных

* Разбить строку по строкам
* Каждую строку разделить по `;`
* Оценки преобразовать в список чисел
* Заполнить словарь студентов

---

### 3. Работа со словарём студентов

* Перебор ключей словаря
* Поиск студента по фамилии
* Добавление и удаление элементов словаря

---

## Меню программы

При запуске программа должна выводить:

```
1. Загрузить данные
2. Показать всех студентов
3. Показать данные о студенте
4. Показать средний балл студента
5. Добавить нового студента
6. Удалить студента
0. Выход из программы
```

---

## Описание пунктов меню

### 1. Загрузить данные

* Парсинг исходной строки
* Создание словаря `students`
* При повторной загрузке данные перезаписываются

---

### 2. Показать всех студентов

* Вывести **список фамилий** всех студентов
* Фамилия — первое слово из ФИО

---

### 3. Показать данные о студенте

* Пользователь вводит фамилию
* Если найден:

  * вывести ФИО
  * группу
  * список оценок
* Если не найден — сообщение об ошибке

---

### 4. Показать средний балл студента

* Поиск студента по фамилии
* Если оценок **5 и более** — вывести средний балл
* Если меньше 5 — вывести строку `Мало оценок`

---

### 5. Добавить нового студента

* Пользователь вводит:

  * ФИО
  * группу
  * оценки
* Новый студент добавляется в словарь

---

### 6. Удалить студента

* Пользователь вводит фамилию
* Если найден — студент удаляется
* Если нет — выводится сообщение

---

### 0. Выход

* Завершение работы программы

---

## ⚠️ Упрощение условия

**Из условия убраны предметы.**
Теперь у каждого студента:

* одна группа;
* один список оценок.

Это снижает уровень вложенности данных и упрощает логику программы.
