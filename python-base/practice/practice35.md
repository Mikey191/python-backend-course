# Контрольная работа

## Тема: Обработка ошибок в Python. Исключения и конструкция try/except/else/finally

### Задание

Дан файл `users.json`, содержащий список пользователей.

Пример структуры данных:

```json
[
    {
        "name": "Bob",
        "email": "bob@gmail.com",
        "login": "bob132",
        "password": "Bob123!"
    }
]
```

Необходимо написать программу для очистки данных пользователей.

---

### Требования

#### 1. Загрузка данных

Создайте функцию:

```python
load_users(filename)
```

которая загружает данные из файла `users.json`.

Если файл отсутствует или содержит некорректный JSON, необходимо обработать ошибку.

---

#### 2. Проверка данных пользователей

Для каждого пользователя необходимо выполнить проверки:

##### Имя (`name`)

* не должно быть пустым;
* должно содержать только буквы.

##### Email (`email`)

* не должен быть пустым;
* обязательно должен содержать символ `@`.

##### Логин (`login`)

* не должен быть пустым;
* может содержать только буквы и цифры.

##### Пароль (`password`)

* не должен быть пустым;
* должен содержать:

  * хотя бы одну букву;
  * хотя бы одну цифру;
  * хотя бы один специальный символ.

При нарушении любого правила необходимо генерировать исключение:

```python
ValueError
```

с понятным текстом ошибки.

---

#### 3. Фильтрация пользователей

Если данные пользователя прошли все проверки:

* добавить пользователя в новый список.

Если обнаружена ошибка:

* записать информацию об ошибке в лог;
* перейти к проверке следующего пользователя.

Программа не должна завершаться из-за одной ошибочной записи.

---

#### 4. Сохранение результата

Создайте функцию:

```python
save_users(filename, users)
```

которая сохраняет корректных пользователей в файл:

```text
clear_users.json
```

---

#### 5. Структура программы

Программа должна содержать следующие функции:

```python
load_users()
save_users()

validate_name()
validate_email()
validate_login()
validate_password()

validate_user()

main()
```

Где:

* четыре функции выполняют отдельные проверки;
* функция `validate_user()` объединяет все проверки;
* функция `main()` управляет всей логикой программы.

---

#### 6. Логирование

Используйте модуль `logging`.

Необходимо фиксировать:

* запуск программы;
* найденные ошибки в данных;
* успешное завершение программы.

---
