### 6.4. –ö–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞: `inherit`, `initial`, `unset`

- `inherit` ‚Äî –ø—Ä–∏–Ω—É–¥–∏—Ç–µ–ª—å–Ω–æ —É–Ω–∞—Å–ª–µ–¥–æ–≤–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –æ—Ç —Ä–æ–¥–∏—Ç–µ–ª—è: `p { color: inherit; }`
- `initial` ‚Äî —Å–±—Ä–æ—Å–∏—Ç—å –∫ –±—Ä–∞—É–∑–µ—Ä–Ω–æ–º—É –∑–Ω–∞—á–µ–Ω–∏—é –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é.
- `unset` ‚Äî –≤–µ–¥—ë—Ç —Å–µ–±—è –∫–∞–∫ `inherit`, –µ—Å–ª–∏ —Å–≤–æ–π—Å—Ç–≤–æ –Ω–∞—Å–ª–µ–¥—É–µ–º–æ, –∏–ª–∏ –∫–∞–∫ `initial`, –µ—Å–ª–∏ –Ω–µ –Ω–∞—Å–ª–µ–¥—É–µ–º–æ.

–ò—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —Ä–µ–¥–∫–æ –Ω–∞ –±–∞–∑–æ–≤–æ–º —É—Ä–æ–≤–Ω–µ, –Ω–æ –ø–æ–ª–µ–∑–Ω—ã –≤ –æ—Ç–¥–µ–ª—å–Ω—ã—Ö —Å–ª—É—á–∞—è—Ö (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Å–±—Ä–æ—Å —Å—Ç–∏–ª–µ–π).

## 7. –°–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å –∏ –∫–∞—Å–∫–∞–¥ (–≤ –¥–≤—É—Ö —Å–ª–æ–≤–∞—Ö)

–ö–æ–≥–¥–∞ –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ —Å–≤–æ–π—Å—Ç–≤–æ –∑–∞–¥–∞–Ω–æ –Ω–µ—Å–∫–æ–ª—å–∫–∏–º–∏ –ø—Ä–∞–≤–∏–ª–∞–º–∏, –±—Ä–∞—É–∑–µ—Ä –≤—ã–±–∏—Ä–∞–µ—Ç, –∫–∞–∫–æ–µ –∏–º–µ–Ω–Ω–æ –ø—Ä–∞–≤–∏–ª–æ –ø—Ä–∏–º–µ–Ω—è—Ç—å, —Ä—É–∫–æ–≤–æ–¥—Å—Ç–≤—É—è—Å—å **–∫–∞—Å—Å–∞–¥–æ–º (Cascading)** –∏ **—Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å—é**.

### 7.1. –ü–æ—Ä—è–¥–æ–∫ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è (—É–ø—Ä–æ—â—ë–Ω–Ω–æ)

1. **–í–∞–∂–Ω–æ (!important):** –ø—Ä–∞–≤–∏–ª–∞ —Å `!important` –∏–º–µ—é—Ç –Ω–∞–∏–≤—ã—Å—à–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç (–Ω–æ –∏—Ö –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –Ω–µ —Ä–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è –±–µ–∑ –∫—Ä–∞–π–Ω–µ–π –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏).
2. **–°–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å:** —á–µ–º ¬´—Ç–æ—á–Ω–µ–µ¬ª —Å–µ–ª–µ–∫—Ç–æ—Ä, —Ç–µ–º –≤—ã—à–µ –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç.
3. **–ò—Å—Ç–æ—á–Ω–∏–∫ –∏ –ø–æ—Ä—è–¥–æ–∫:** –µ—Å–ª–∏ —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å –æ–¥–∏–Ω–∞–∫–æ–≤–∞—è ‚Äî –ø–æ–±–µ–∂–¥–∞–µ—Ç –ø—Ä–∞–≤–∏–ª–æ, –∫–æ—Ç–æ—Ä–æ–µ –∏–¥—ë—Ç –Ω–∏–∂–µ –≤ CSS (–ø–æ—Å–ª–µ–¥–Ω–µ–µ –æ–±—ä—è–≤–ª–µ–Ω–Ω–æ–µ).

### 7.2. –ö–∞–∫ —Å—á–∏—Ç–∞–µ—Ç—Å—è —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å (—É–ø—Ä–æ—â—ë–Ω–Ω–∞—è —à–∫–∞–ª–∞)

- inline-—Å—Ç–∏–ª–∏ (`style="..."`) ‚Äî —Å–∞–º–∞—è –≤—ã—Å–æ–∫–∞—è ¬´—Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å¬ª (–ø—Ä–∏–º–µ—Ä–Ω–æ 1000 –≤ —É—Å–ª–æ–≤–Ω–æ–π —Å–∏—Å—Ç–µ–º–µ).
- —Å–µ–ª–µ–∫—Ç–æ—Ä `id` (`#id`) ‚Äî –±–æ–ª—å—à–æ–π –≤–µ—Å (–ø—Ä–∏–º–µ—Ä–Ω–æ 100).
- —Å–µ–ª–µ–∫—Ç–æ—Ä –∫–ª–∞—Å—Å–∞, –∞—Ç—Ä–∏–±—É—Ç–∞, –ø—Å–µ–≤–¥–æ–∫–ª–∞—Å—Å–∞ (`.class`, `[attr]`, `:hover`) ‚Äî —Å—Ä–µ–¥–Ω–∏–π –≤–µ—Å (–ø—Ä–∏–º–µ—Ä–Ω–æ 10).
- —Å–µ–ª–µ–∫—Ç–æ—Ä —Ç–µ–≥–∞/–ø—Å–µ–≤–¥–æ—ç–ª–µ–º–µ–Ω—Ç–∞ (`h1`, `p`, `::before`) ‚Äî –Ω–∏–∑–∫–∏–π –≤–µ—Å (–ø—Ä–∏–º–µ—Ä–Ω–æ 1).

**–ü—Ä–∏–º–µ—Ä—ã:**

- `p { color: red; }` ‚Äî —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å 1
- `.intro p { color: blue; }` ‚Äî —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å 10 + 1 = 11
- `#main .intro p { color: green; }` ‚Äî —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å 100 + 10 + 1 = 111

–ï—Å–ª–∏ –¥–≤–∞ –ø—Ä–∞–≤–∏–ª–∞ –∫–æ–Ω—Ñ–ª–∏–∫—Ç—É—é—Ç, –±—Ä–∞—É–∑–µ—Ä –≤—ã–±–µ—Ä–µ—Ç –ø—Ä–∞–≤–∏–ª–æ —Å –±–æ–ª—å—à–µ–π —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å—é. –ï—Å–ª–∏ —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç–∏ —Ä–∞–≤–Ω—ã ‚Äî –ø—Ä–∞–≤–∏–ª–æ, —É–∫–∞–∑–∞–Ω–Ω–æ–µ –ø–æ–∑–∂–µ, –ø–æ–±–µ–∂–¥–∞–µ—Ç.

### 7.3. –ü—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏–π –ø—Ä–∏–º–µ—Ä –∫–æ–Ω—Ñ–ª–∏–∫—Ç–∞


–ï—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç `<p>` –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –≤–Ω—É—Ç—Ä–∏ `#main` –∏ `.intro`, –ø—Ä–∏–º–µ–Ω–∏—Ç—Å—è `color: green` (–ø–æ—Ç–æ–º—É —á—Ç–æ `#main p` –±–æ–ª–µ–µ —Å–ø–µ—Ü–∏—Ñ–∏—á–µ–Ω).

### 7.4. –ö–æ–≥–¥–∞ –ø—Ä–∏–º–µ–Ω—è—Ç—å —á—Ç–æ

- –ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ —Å–µ–ª–µ–∫—Ç–æ—Ä—ã —Ç–µ–≥–æ–≤ –¥–ª—è –≥–ª–æ–±–∞–ª—å–Ω—ã—Ö, –±–∞–∑–æ–≤—ã—Ö –ø—Ä–∞–≤–∏–ª.

- ID ‚Äî —Ç–æ–ª—å–∫–æ –¥–ª—è —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö —Å–ª—É—á–∞–µ–≤ (–∏ –æ—Å—Ç–æ—Ä–æ–∂–Ω–æ, —Ç.–∫. –ø–æ–≤—ã—à–∞–µ—Ç —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å).
- –ù–µ –∑–ª–æ—É–ø–æ—Ç—Ä–µ–±–ª—è–π—Ç–µ `!important`.

## 8. –°—Ç–∏–ª–∏–∑–∞—Ü–∏—è —Å–µ–º–∞–Ω—Ç–∏—á–µ—Å–∫–æ–π —Ä–∞–∑–º–µ—Ç–∫–∏ —Å –ø–æ–º–æ—â—å—é —Ä–∞—Å—à–∏—Ä–µ–Ω–Ω—ã—Ö —Å–µ–ª–µ–∫—Ç–æ—Ä–æ–≤ ‚Äî –ø—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏–µ –ø—Ä–∏–º–µ—Ä—ã

–¢–µ–ø–µ—Ä—å ‚Äî –æ–±—ä–µ–¥–∏–Ω—è–µ–º –≤—Å—ë: **—Å–µ–º–∞–Ω—Ç–∏–∫–∞ + —Ä–∞—Å—à–∏—Ä–µ–Ω–Ω—ã–µ —Å–µ–ª–µ–∫—Ç–æ—Ä—ã + –Ω–∞—Å–ª–µ–¥–æ–≤–∞–Ω–∏–µ + —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å**. –ù–∏–∂–µ ‚Äî –Ω–∞–±–æ—Ä –ø–æ–ª–µ–∑–Ω—ã—Ö –ø—Ä–∏–º–µ—Ä–æ–≤ (–≤–∑—è—Ç—ã –∏–∑ —à–∞–±–ª–æ–Ω–∞, –∫–æ—Ç–æ—Ä—ã–π —Ä–∞–Ω–µ–µ —Å–æ–±–∏—Ä–∞–ª–∏).

### –ü—Ä–∏–º–µ—Ä –±–∞–∑–æ–≤–æ–π —Ä–∞–∑–º–µ—Ç–∫–∏ (–Ω–∞–ø–æ–º–∏–Ω–∞–µ–º)

### –ü—Ä–∏–º–µ—Ä CSS —Å —Ä–∞—Å—à–∏—Ä–µ–Ω–Ω—ã–º–∏ —Å–µ–ª–µ–∫—Ç–æ—Ä–∞–º–∏ –∏ –ø–æ—è—Å–Ω–µ–Ω–∏—è–º–∏

### –ü–æ—è—Å–Ω–µ–Ω–∏—è –∏ —Å–æ–≤–µ—Ç—ã –ø–æ —ç—Ç–æ–º—É CSS

## –ù–µ–±–æ–ª—å—à–∏–µ –ø—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏–µ –∑–∞–¥–∞–Ω–∏—è
